# About the Project 
- eart disease is one of the leading causes of death worldwide. Early detection can help healthcare professionals make better clinical decisions and improve patient outcomes. The purpose of this project is to analyze a heart disease dataset, discover meaningful patterns through Exploratory Data Analysis (EDA), and build machine learning models capable of predicting whether a patient has heart disease.

This project follows a complete data analysis workflow, starting from data exploration and cleaning, moving through visualization and feature selection, and ending with classification model development and evaluation.

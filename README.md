# About the Project 
- eart disease is one of the leading causes of death worldwide. Early detection can help healthcare professionals make better clinical decisions and improve patient outcomes. The purpose of this project is to analyze a heart disease dataset, discover meaningful patterns through Exploratory Data Analysis (EDA), and build machine learning models capable of predicting whether a patient has heart disease.

- This project follows a complete data analysis workflow, starting from data exploration and cleaning, moving through visualization and feature selection, and ending with classification model development and evaluation.

# Objectives

-   Explore and understand the dataset.
-   Clean and preprocess the data.
-   Perform EDA and visualize key insights.
-   Build and compare classification models.
-   Evaluate model performance.

# Dataset

-   Dataset: Heart Disease Cleveland UCI
-   Records: 297
-   Features: 14
-   Target: Heart Disease (0 = No Disease, 1 = Disease)

# Models

-   Logistic Regression
-   Decision Tree

# Results

-   Logistic Regression: 71.67%
-   Decision Tree: 66.67%

Logistic Regression achieved the best performance.

# Key Insights

-   The dataset is relatively balanced.
-   Older patients are more likely to have heart disease.
-   Chest pain type is an important indicator.
-   Maximum heart rate has a negative relationship with heart disease.
-   Logistic Regression outperformed Decision Tree.
